## 🎮 关于本工具

**PartyFish** 是专门为《猛兽派对 (Party Animals)》中的**钓鱼小游戏**
设计的自动化辅助工具。猛兽派对的钓鱼玩法需要玩家反复进行抛竿、等待咬钩、拉杆等操作，本工具可以帮助你自动完成这些重复性操作。

---

## ✨ 功能特性

### 功能

- 🐟️ **自动钓鱼** - 掏出鱼竿后按f2即可自动钓鱼
- 🌊 **自动放鱼** - 自动钓鱼过程中钓满了会自动钓鱼(可配置丢弃鱼品质)，按f3主动丢鱼 ```温馨提示:丢鱼过程中不要手动操作鼠标```
- 📊 **钓鱼记录系统** - 自动记录钓鱼成果，支持按品质筛选和搜索
- 📊 **每种品质鱼统计** - 计算本次以及历史统计
- 🚫 **自动挂机** - 可自动挂机保证不被系统踢出
- 🎮 **uno刷牌自动跳过** - 按f5可自动跳过(可配置是否一直跳过，以及跳过至指定牌数)

### 特性

- 🖥️ **多分辨率支持** - 支持 1080P、2K、4K 及自定义分辨率 ```第一次启动自动识别系统分辨率```
- 🎯 **智能图像识别** - 基于 OpenCV 模板匹配，精准识别游戏状态
- 🔤 **OCR 识别** - 使用 RapidOCR 识别鱼的名称、品质和重量
- 💾 **参数持久化** - 自动保存配置参数，下次启动自动加载
- 🎨 **现代化界面** - 基于 ttkbootstrap 的深色主题 GUI
- 🧵 **多线程架构** - 主逻辑后台运行，GUI 响应流畅

```bash

```

#### 2.将``config``目录下的``parameters.json``文件(包含目录)和``resources``资源目录拷贝至``dist``目录

#### 3.运行``dist`目录下的``AutoFish.exe``

## 📖 使用方法

1. 启动程序后，会显示 GUI 配置界面
2. 根据猛兽派对的游戏设置调整参数（或使用默认值）
3. 切换到猛兽派对游戏窗口，确保钓鱼界面可见
4. 按 **F2** 键（或自定义热键）启动自动钓鱼
5. 再次按热键暂停

## ⚙️ 参数配置说明

### 钓鱼参数

| 参数     | 默认值   | 说明           |
|--------|-------|--------------|
| 鱼杆类型   | 路亚重竿  | 一共能保存6种竿参数   |
| 循环间隔   | 0.3 秒 | 主循环检测间隔时间    |
| 收线时间   | 0.9 秒 | 鼠标左键按下持续时间   |
| 放线时间   | 0.3 秒 | 鼠标左键释放后等待时间  |
| 最大拉杆次数 | 50 次  | 单次钓鱼最大拉杆循环次数 |
| 抛竿时间   | 0.5 秒 | 抛竿时鼠标按下持续时间  |
| 时间抖动范围 | 15%   | 适当的延迟        |

---

### 加时选项

- **是**: 自动点击加时按钮
- **否**: 自动点击不加时按钮

---

### 丢鱼参数

| 参数     | 默认值 | 说明                 |
|--------|-----|--------------------|
| 自动丢鱼开关 | 开启  | 鱼钓满时或主动按f3时可触发自动丢鱼 |
| 丢鱼品质阈值 | 史诗  | 默认丢弃史诗以下品质鱼(史诗保留)  |

---

### uno参数

| 参数     | 默认值 | 说明                |
|--------|-----|-------------------|
| 是否一直跳过 | 否   | 帮其他玩家刷牌时可将此参数改为是  |
| 抽取牌数   | 35  | 主动刷牌时 摸完35张牌后脚本暂停 |

---

### 分辨率设置

程序默认以 **2K (2560×1440)** 为基准分辨率设计```第一次打开程序会自动读取当前系统尺寸```，支持以下预设：

| 分辨率       | 尺寸          |
|-----------|-------------|
| **1080P** | 1920 × 1080 |
| **2K**    | 2560 × 1440 |
| **4K**    | 3840 × 2160 |
| **自定义**   | 手动输入宽高      |

## 🔧 工作原理

- **鱼饵数量检测**: 通过模板匹配识别屏幕上的鱼饵数量
- **状态变化检测**: 当鱼饵数量减少时，判断为有鱼上钩
- **自动拉杆**: 执行收线/放线循环操作
- **上鱼检测**: 识别星星图标判断钓鱼成功
- **OCR 识别**: 识别捕获的鱼的信息并记录
- **加时检测**: 通过模板匹配识别屏幕上`长`
- **空桶检测**: 通过模板匹配识别屏幕上出现`空空如也`，桶中没有鱼时暂停丢鱼
- **桶满检测**: 通过模板匹配识别屏幕上出现`桶已钓满`，桶满时触发自动丢鱼
- **桶中第一条鱼是否锁住检测**: 通过模板匹配识别屏幕上出现`🔒`
- **桶中48条鱼检测**: 通过模板匹配识别屏幕上出现`48/48`，桶中鱼满48条且都被锁住是暂停自动钓鱼、放鱼
- **鱼品质检测**: 通过模板匹配识别屏幕上指定像素点，(桶满或f3触发)检测当前鱼是应该丢弃还是锁定
- **uno跳过检测**: 通过模板匹配识别屏幕上跳过按钮

## ⚠️ 注意事项

1. **管理员权限**: 猛兽派对可能需要以管理员身份运行本程序
2. **全屏模式**: 建议猛兽派对使用全屏模式
3. **分辨率匹配**: 请确保选择与猛兽派对实际分辨率匹配的设置
4. **杀毒软件**: 如遇杀毒软件误报，请添加信任或关闭杀软后使用

## 🛠️ 故障排除

| 问题      | 解决方案           |
|---------|----------------|
| 程序无法启动  | 尝试以管理员身份运行     |
| 未识别到鱼饵  | 检查分辨率设置是否正确    |
| 无法抛竿    | 确保猛兽派对游戏窗口处于前台 |
| 杀毒软件拦截  | 添加程序到白名单       |
| GUI 无响应 | 检查是否有其他程序占用资源  |
